Esto es un proyecto creado en Angular 16 para aprender a utilizar el framework. El proyecto consiste en una aplicación web que permite almacenar contraseñas de forma segura. Para ello, se utiliza una base de datos en la nube.

Funcionalidades

  • Crear contraseñas
  • Modificar contraseñas
  • Borrar contraseñas
  • Visualizar contraseñas
  • Copiar contraseñas al portapapeles
  • Autenticación de usuarios
