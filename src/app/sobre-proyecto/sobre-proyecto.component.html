KeyCloud - Gestor Seguro de Contraseñas

KeyCloud es una aplicación web desarrollada en Angular 16 que permite gestionar contraseñas de forma segura. La API y la base de datos operan como un microservicio, garantizando una arquitectura flexible y escalable.

Características Principales

  • CRUD completo de contraseñas (Crear, Leer, Actualizar y Eliminar).
  • Almacenamiento seguro con cifrado de contraseñas en la base de datos y en las transferencias.
  • Autenticación de usuarios con gestión de acceso personalizado.
  • Generación automática de contraseñas seguras.
  • Indicador visual de fortaleza de contraseñas.
  • Marcado de contraseñas próximas a caducar.
  • Opción para copiar contraseñas al portapapeles con un doble clic.
